===========================
Formulário
===========================

Use formulários para permitir que as pessoas insiram dados para usar na aplicação ou para configurar opções. 
O design de formulários requer decisões sobre estrutura, sequência, elementos de interface, rótulos de campo, ajuda e feedback.
